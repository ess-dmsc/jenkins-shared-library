          Create a map of builders to be passed to a Jenkins parallel step. Standard Jenkins pipeline steps are executed on the allocated build node outside the container. To run commands with the container, call the Container methods on the closure parameter.

          Parameters:

          Get a Jenkins pipeline stage with automated failure messages. If an exception occurs inside the stage block, the messages are saved for handleFailureMessages(), identifying the stage name.

          throws:

          Parameters:
          name - stage name
          stageCommands - block of commands between curly braces
